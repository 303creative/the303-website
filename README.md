# 303 Creative Agency®

Sitio web oficial de 303 Creative Agency — Branding & Marketing Strategy.

## Estructura

```
the303-vercel/
├── public/
│   ├── index.html      ← Página principal
│   ├── logo.jpg         ← Logo (fondo blanco)
│   └── logo.png         ← Logo (fondo negro)
├── vercel.json          ← Configuración de Vercel
├── package.json         ← Metadata del proyecto
└── README.md            ← Este archivo
```

## Deploy

Este proyecto se despliega automáticamente en Vercel con cada push a `main`.

## Desarrollo

Para agregar páginas nuevas, crear archivos HTML en la carpeta `public/`.
Por ejemplo: `public/about.html`, `public/services.html`, etc.
